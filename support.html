STUCK?

Email hello@takeoff.llc. A real human (probably the dad who built this) will write back. Usually within 24 hours.

COMMON FIXES

Order didn't show up on the kitchen iPad?
Both devices need to be on the same family code. Check the bar at the top of the screen — same code on both? If yes, force-close and reopen Bella's Kitchen on the kitchen iPad.

App icon won't add to home screen on iPhone?
Open the site in Safari (not Chrome). Tap the Share button → Add to Home Screen.

XP didn't go up after I cooked an order?
Make sure you have a name set under Crew. XP attaches to whoever is signed into that device.
