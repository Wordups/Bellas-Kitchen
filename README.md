# Bella's Kitchen

Family dinner, made fun. Kids run the order pad, you run the kitchen, the app
keeps the line moving.

**Live:** https://wordups.github.io/Bellas-Kitchen/ (soon: https://bellaskitchen.app)

## What it does

- **Order pad** — tap-driven chip grid across 5 meal modes (sandwich, taco,
  breakfast, pizza, build-a-bowl). Receipt prints, kid hits *Text Dad*, the
  order shows up in Messages with everything pre-filled.
- **Kitchen panel** — three-lane line (Pending / Cooking / Ready) that updates
  in realtime when a phone in the family submits. Tap a ticket to advance it.
  Late tickets go red and pulse.
- **The Crew** — XP, streaks, badges. Take an order +10, cook +15, deliver +5,
  3-day streak +25. Six badges to chase.

## How it's built

- Vanilla JS ESM, no framework. Static-hosted on GitHub Pages.
- **Local mode** (default): `localStorage` + `BroadcastChannel` for cross-tab
  realtime on a single device. Works the moment you open the page.
- **Cloud mode** (opt-in): drop Supabase credentials into `js/config.js` and
  the app upgrades to multi-device realtime via Supabase Postgres + Realtime.
- PWA with offline-capable service worker (`service-worker.js`), maskable
  icons, app shortcuts.
- Capacitor scaffolded for an iOS App Store build (requires a Mac).

## Working on it

This is a pure static site — no build step. Edit any file, open `index.html`
in a browser, you're done.

To regenerate icons after a brand tweak: `python make_icons.py`.

To turn on cloud sync, set up a Supabase project and copy `js/config.example.js`
to `js/config.js`. See [SETUP.md](SETUP.md) for the schema + checklist.
